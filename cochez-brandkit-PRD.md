## 1. Resumen Ejecutivo

**Cochez y Cía, S.A.** es la cadena de materiales de construcción, mejoramiento del hogar, ferretería y acabados líder en Panamá. Con más de 60 años de trayectoria, 23+ sucursales a nivel nacional y más de 323,000 seguidores en Instagram, la marca representa solidez, confianza y cercanía con el panameño.

Este PRD define todos los elementos de identidad necesarios para construir una **página de Brand Kit** coherente, usable y fiel a la marca, que sirva como referencia oficial para equipos internos, agencias y proveedores.

## 5. Identidad Visual

### 5.1 Logo

#### Descripción
El logotipo de **Cochez y Cía** utiliza un wordmark tipográfico en letras mayúsculas o caja alta, con el texto "COCHEZ" como elemento principal y "y Cía" como elemento secundario/descriptor. La marca es reconocible sin necesidad de ícono, aunque puede acompañarse de elementos gráficos de soporte.

#### Reglas de Uso del Logo

| Regla | Especificación |
|---|---|
| Versión principal | Sobre fondo blanco o claro |
| Versión invertida | Logo blanco sobre fondo de color de marca |
| Espacio mínimo libre | Equivalente a la altura de la "C" del logotipo en cada lado |
| Tamaño mínimo digital | 120px de ancho |
| Tamaño mínimo impreso | 25mm de ancho |
| Usos prohibidos | Distorsionar, rotar, cambiar colores, agregar sombras, usar sobre fondos con poco contraste |

#### Formatos a Incluir en Brand Kit
- `.SVG` — Logo principal (color)
- `.SVG` — Logo invertido (blanco)
- `.SVG` — Logo monocromático (negro)
- `.PNG` — Alta resolución (fondo transparente, 2x y 3x)
- `.PDF` — Para uso en imprenta

#### Tipografía Secundaria (Body & UI Text)

| Uso | Familia | Peso | Tamaño referencial |
|---|---|---|---|
| Párrafos / Cuerpo | **Open Sans** | Regular (400) | 16px / 1.6 line-height |
| Texto UI (botones, labels) | **Open Sans** | SemiBold (600) | 14–16px |
| Texto legal / Footnotes | **Open Sans** | Regular (400) | 12–13px |
| Precios / Números destacados | **Montserrat** | ExtraBold (800) | 24–40px |

#### Escala Tipográfica (sistema 4pt)

```
64px — Hero / Super headline
48px — H1
40px — H2
32px — H3
24px — H4 / Large label
20px — Lead / Intro text
16px — Body text (base)
14px — Small / UI labels
12px — Caption / Legal
```

#### LineHeight recomendado

| Nivel | Line Height |
|---|---|
| Headings (H1–H3) | 1.2 |
| H4 / Subtítulos | 1.3 |
| Body text | 1.6 |
| Captions | 1.4 |

### 5.4 Iconografía y Elementos Gráficos

#### Estilo de Iconos
- **Estilo:** Line icons con trazo de 2px, bordes redondeados (rounded caps/joins)
- **Grid base:** 24×24px
- **Librería recomendada:** Material Symbols (Google) o Phosphor Icons
- **Color:** Heredado del contexto (naranja sobre blanco, blanco sobre naranja, gris oscuro sobre fondos claros)

#### Elementos Decorativos de Marca
- Ángulos y formas geométricas anguladas (evocando construcción, planos)
- Texturas sutiles inspiradas en materiales (cemento, madera, metal)
- Líneas diagonales como elemento gráfico de energía y movimiento
- Badges circulares para ofertas (estilo "precio especial", "% off")

#### Fotografía
| Tipo | Guía |
|---|---|
| **Lifestyle** | Familias en proceso de renovación del hogar, alegría y calidez |
| **Producto** | Fondo blanco o gris claro, iluminación limpia, sin sombras duras |
| **Profesional (RedPro)** | Trabajadores en obra, look auténtico y no ilustrativo |
| **Materiales** | Macro shots de texturas, acabados, calidad |
| **Sostenibilidad** | Espacios naturales, paneles solares, verde |
| **Sucursales** | Amplias, organizadas, accesibles, modernas |

#### Tono fotográfico
- Cálido, saturación media-alta
- Luz natural o simulada natural
- Evitar: filtros vintage, tonos fríos, imágenes de stock genéricas sin contexto panameño

## 6. Arquitectura de Marca (Brand Architecture)

Cochez y Cía opera un ecosistema de submarcas:

```
COCHEZ Y CÍA (marca madre)
├── PuntoCochez         — Formato especializado (pinturas + herramientas)
├── Kohler by Cochez    — Tienda premium de plomería y diseño
├── RedPro              — Programa para profesionales de la construcción
├── PuntoCochez Loyalty — Programa de puntos "Puntos Gordos"
└── Encomiendas Cochez  — Servicio de envío de paquetes entre sucursales
```

### Reglas de Submarca

| Submarca | Color principal | Tipografía | Relación con marca madre |
|---|---|---|---|
| PuntoCochez | Naranja Cochez + Verde | Montserrat | Claramente asociada a Cochez |
| Kohler by Cochez | Negro / Plata (Kohler) + Naranja Cochez | Montserrat | Cobranding — Kohler predomina |
| RedPro | Azul Profesional + Naranja Cochez | Montserrat | Submarca de Cochez |
| Encomiendas | Naranja Cochez + Gris | Montserrat + Open Sans | Servicio de Cochez |

## 9. Requisitos de la Brand Kit Page

### 9.1 Objetivo de la Página
Crear una página web accesible internamente (y posiblemente pública) que centralice todos los activos de marca de Cochez y Cía para uso de equipos internos, agencias de publicidad, medios de comunicación y proveedores.

### 9.2 Secciones Requeridas de la Página

#### Sección 1: Introducción de Marca
- Nombre oficial y logo en sus variantes
- Propósito, Misión, Visión en formato visual
- Arquetipos y valores en íconos

#### Sección 2: Logos y Activos
- Visualización de todas las variantes del logo
- Botones de descarga por formato (SVG, PNG, PDF)
- Do's & Don'ts visuales del logo

#### Sección 3: Paleta de Colores
- Swatches interactivos mostrando HEX, RGB, CMYK
- Botón "copiar al portapapeles" por color
- Ejemplos de aplicación en contexto

#### Sección 4: Tipografía
- Specimen de cada tipografía (todas las variantes de peso)
- Escala tipográfica visual
- Descarga o link a Google Fonts

#### Sección 5: Iconografía
- Grid de íconos aprobados
- Guía de estilo de iconos
- Descarga del icon set

#### Sección 6: Fotografía
- Ejemplos de fotografía aprobada por categoría
- Guía de tono fotográfico (Do's & Don'ts)
- Acceso a biblioteca de imágenes (si aplica)

#### Sección 7: Voz y Tono
- Personalidad de marca
- Tabla de tono por contexto
- Ejemplos de textos correctos vs. incorrectos

#### Sección 8: Plantillas y Recursos
- Links de descarga a plantillas (presentaciones, social media, email)
- Acceso a Figma / Canva Brand Kit (si aplica)

#### Sección 9: Arquitectura de Marca / Submarcas
- Diagrama del ecosistema
- Guía de cada submarca

#### Sección 10: Contacto
- Equipo de Marca / Marketing para consultas
- Proceso de aprobación de materiales

### 9.3 Especificaciones Técnicas de la Página

| Especificación | Requerimiento |
|---|---|
| Framework | React + Tailwind CSS (recomendado) |
| Responsive | Mobile-first, breakpoints: 768px, 1024px, 1440px |
| Navegación | Sticky sidebar (desktop) / menú hamburguesa (mobile) |
| Accesibilidad | WCAG 2.1 AA — contraste mínimo 4.5:1 en texto |
| Rendimiento | LCP < 2.5s, CLS < 0.1 |
| Interactividad | Copy-to-clipboard en códigos de color, preview de logo en dark/light |
| Descarga | Todos los activos en `.zip` individual y pack completo |
| Autenticación | Opcional — acceso público o protegido por contraseña |

### 9.4 Componentes UI Clave para la Página

```
ColorSwatch        — Muestra nombre, HEX, RGB, CMYK + copy button
LogoCard           — Preview logo + variantes + download button
FontSpecimen       — Muestra tipografía en todos sus pesos con texto de muestra
DoAndDont          — Dos columnas: uso correcto vs. incorrecto
IconGrid           — Galería de íconos descargables
PhotoExample       — Grid de fotografía categorizada
ToneTable          — Tabla de tono por contexto
BrandArchDiagram   — Diagrama visual del ecosistema de marcas
```
